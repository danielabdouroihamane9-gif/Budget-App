# 💰 Budget App

A simple and efficient **Python** application to help track personal finances.

## 🚀 Features
- Track daily expenses and income.
- Categorize spending (Food, Rent, Fun, etc.).
- View total balance and monthly summaries.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Storage:** CSV/JSON (or whatever you plan to use)

## 📖 How to Use
1. Clone the repo.
2. Run `python main.py`.
